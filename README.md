# Termidoro 🍅

A minimalist, keyboard-driven Animedoro timer for the Linux terminal. Built using (just) Python and Textual.

## Why does this even exist bro?

Since I'm using linux (I use arch btw), it doesn't come with a timer app. Whenever I work on something like coding, I lost track of time which isn't really good sincee I also want to do other stuff apart from coding. I created this simple app over the weekend to solve this personal problem of mine. ALSO, I'm sure that there are other timers out there it's just fun building stuff.

## Features Included

- **Study Mode:** 40-minute focus sessions. (other any other work)
- **Anime Mode:** 20-minute break sessions (Anything other than anime is fine its just for break).
- **Keyboard Driven:** Everything is controlled with your keyboard. as a linux app should be. (Instructions will be on the footer when opening the app)

## Installation

Ensure you have [uv](https://github.com/astral-sh/uv) installed.

```
git clone https://github.com/KisukeMaybe/Termidoro.git
cd termidoro
uv run main.py
```
